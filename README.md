# monitoring-infra-qgis-plugins

생활인프라 모니터링을 위한 QGIS 플러그인 모음입니다.
시군구 단위로 공공시설의 **공급수준 · 향유수준 · 충족수준**을 산출하고, 이를 종합한 **생활인프라 편리성 종합지수**를 계산합니다.

**제작**: 손종혁 (국토연구원 국토모니터링연구센터)

---

## 플러그인 구성

| # | 폴더 | 플러그인명 | 설명 |
|---|------|-----------|------|
| 2 | `monitoring_qgis_plungin_supply_index_gpd` | 공급수준 분석 | 시설 Point SHP × 인구 SHP → 1천인당 시설 수 → 부문별 공급수준 |
| 3 | `monitoring_qgis_plungin_service_pop` | 향유수준 분석 | 서비스권역 내 인구비율 SHP → 부문별 향유수준 |
| 4 | `monitoring_qgis_plungin_access_idx` | 충족수준 분석 | 격자별 접근성 거리 SHP → 충족 격자 비율 → 부문별 충족수준 |
| 5 | `monitoring_qgis_plungin_composite` | 생활인프라 편리성 종합지수 | 2·3·4번 결과를 가중 합산 → 0~100 종합지수 |

---

## 분석 체계

```
공급수준 (2번)  ─┐
향유수준 (3번)  ─┼─ 부문별 가중 합산 ─► 교육학습 편리성
충족수준 (4번)  ─┘                      돌봄복지 편리성
                                        보건의료 편리성    ─► 생활인프라
                                        안전치안 편리성         편리성
                                        체육문화 편리성      종합지수
                                                            (0~100점)
```

5개 부문: **교육학습 · 돌봄복지 · 보건의료 · 안전치안 · 체육문화**

## 설치 방법

1. 각 플러그인 폴더를 QGIS 플러그인 디렉토리에 복사합니다.
   - Windows: `C:\Users\{사용자명}\AppData\Roaming\QGIS\QGIS3\profiles\default\python\plugins\`
2. QGIS 재시작 후 **플러그인 관리자**에서 활성화합니다.

### 의존성 설치 (geopandas)

플러그인 2·3·4·5번은 `geopandas`가 필요합니다.
OSGeo4W Shell에서 아래 명령어를 실행하세요.

```bash
pip install geopandas
```

## 플러그인별 입력 데이터

### 공급수준 (2번)
- Point SHP 폴더: 시설 위치 포인트 (시설명이 파일명에 포함)
- 인구 SHP: 시군구 경계 + 인구 컬럼 포함

### 향유수준 (3번)
- 서비스권역 SHP 폴더: `value_r` 컬럼(서비스권역 내 인구비율) + `sgg_cd` 포함
- 파일명 패턴: `숫자.숫자 시설명(시군구) ...`

### 충족수준 (4번)
- 접근성 SHP 폴더: `value` 컬럼(접근성 거리, km) + `gid`, `sgg_cd` 포함
- 시군구 경계 SHP: 최종 시군구 단위 집계용
- 파일명 패턴: `숫자.숫자 시설명(시군구격자) 접근성`

### 종합지수 (5번)
- 공급수준 결과 SHP (`supply_index.shp`)
- 향유수준 결과 SHP (`service_pop_index.shp`)
- 충족수준 결과 SHP (`access_index.shp`)

## 출력 컬럼

### 공급수준 · 향유수준
| 컬럼 패턴 | 설명 |
|-----------|------|
| `{fac}_cnt` | 시설 개수 (공급수준만) |
| `{fac}_sup` | 1천인당 시설 수 (공급수준만) |
| `{fac}_std` | 표준화 값 |
| `{sec}_avg` | 부문별 평균 (edu / care / med / safe / cult) |

### 충족수준
| 컬럼 패턴 | 설명 |
|-----------|------|
| `{sec}_ok`  | 충족 격자 수 |
| `{sec}_rat` | 충족 격자 비율 |
| `{sec}_std` | 표준화 값 |

### 종합지수
| 컬럼 | 설명 |
|------|------|
| `edu_conv` | 교육학습 편리성 |
| `care_conv` | 돌봄복지 편리성 |
| `med_conv` | 보건의료 편리성 |
| `safe_conv` | 안전치안 편리성 |
| `cult_conv` | 체육문화 편리성 |
| `infra_raw` | 생활인프라 편리성 원값 |
| `infra_idx` | 생활인프라 편리성 종합지수 (0~100) |
